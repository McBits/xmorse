Learn Morse Code

System requirements: A browser that supports the Web Audio API, such as Chrome, Firefox, or Edge.

Known issue: A lot of mobile devices struggle with audio timing at the precision needed for Morse code, so this site is only usable up to about 8-10 WPM. I'm still looking for a solution...

If you have never learned the Morse code characters, listen with the voice enabled. Try to remember the character before the voice reveals it. Once you have memorized most of the sequences, you can disable the voice. (This takes less time than you might think!)

Set the character WPM just faster than your target speed, and set the spacing just wide enough to pose a challenge. For example, if you are aiming for 18 WPM, listen to characters at 20-22 WPM.

Gradually reduce the spacing between characters until you reach your target rate.

Practice "copying" (writing or typing) characters while you listen. This will reinforce your learning better than simply listening.

Set aside a few minutes every day to practice. It's the small, consistent efforts that add up to long-term success!
