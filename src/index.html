xMorse

Learning Morse code has never been easier.

Press Start to begin playing Morse code characters.

Some tips for success:

  • Learn the characters. Listen with the voice enabled, and try to anticipate the character before the voice says it. Once you have memorized the Morse code sequences, you can disable the voice. (This takes less time than you would think!)
  • Set the character WPM just faster than your target speed, and set the spacing just wide enough to pose a challenge. For example, if you are aiming for 18 WPM, listen to characters at 20-22 WPM.
  • Gradually reduce the spacing between characters until you reach your target rate.
  • Practice "copying" (writing or typing) characters as you listen. This will reinforce your learning more than listening alone.
  • Set aside 10 minutes a day to practice. It's the small, consistent efforts that add up to long-term success.
